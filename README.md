# Visible Showcase

Visible is a business discovery and online booking platform built for local businesses in Kosovo.

The platform allows users to discover businesses by city and category, view business profiles, check contact details, browse galleries, and create online bookings. Businesses can manage their booking requests through a dedicated dashboard.

> Source code is private because this is a commercial project. This repository is used only as a portfolio showcase.

## Live Project

https://visible-ks.com

## Tech Stack

- Laravel
- PHP
- MySQL
- Blade
- Tailwind CSS
- JavaScript
- Authentication
- Email notifications
- Multilingual support

## Key Features

- Business discovery by category and city
- Homepage with search and popular businesses
- Category-based business result pages
- Business profiles with contact details, gallery, reviews, and map links
- Online booking flow for businesses
- Business dashboard for managing booking requests
- Booking statuses such as pending, accepted, and rejected
- Multilingual interface in Albanian and English
- Admin-side management for businesses, categories, cities, and content
- Responsive layout for desktop and mobile users
